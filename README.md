# sl10-sec.github.io

Personal scratchpad for offensive security research, notes, CTF writeups, and interesting reads.

Built with [Jekyll](https://jekyllrb.com/) and the [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) theme, hosted on GitHub Pages.

## Content structure

| Folder | Section |
|---|---|
| `_daily/` | Daily Read |
| `_musings/` | Musings |
| `_research_notes/` | Research Notes |
| `_writeups/` | Write-ups |
